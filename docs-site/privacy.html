1. What Data We Collect

AI Code Buddy collects minimal data, and all of it stays on your device:

üìù Configuration Data (Stored Locally)

  • AI Provider Selection - Which provider you chose (Claude, OpenAI, etc.)
  • API Keys - Stored encrypted in Chrome's local storage
  • Model Preferences - Which model you selected (e.g., GPT-4, Claude Sonnet)
  • Custom Settings - Temperature, max tokens, custom prompts, CSS selectors
  • Extension Preferences - UI settings, auto-improve toggle, etc.
‚ö†Ô∏è Important: ALL of this data is stored locally on your device using Chrome's chrome.storage.local API. It is never transmitted to Apra Labs.

2. What We Do NOT Collect

We are committed to your privacy. We explicitly DO NOT collect:

  • ‚ùå Personal Information - No name, email, phone, address

3. How Data is Stored

üîê Chrome's Encrypted Storage

Your API keys and settings are stored using Chrome's built-in chrome.storage.local API:

  • Encrypted at rest - Chrome automatically encrypts storage
  • Isolated to extension - Other extensions can't access it
  • Syncs with Chrome Sync - Only if you enable Chrome sync (optional)
  • Cleared when extension removed - Uninstalling removes all data

üåê Network Transmission

When you use AI Code Buddy, here's exactly what happens:

Data Flow:

  1. You click "Send to AI" on terminal output
  2. Extension retrieves your API key from local storage
  3. Extension sends request directly to your chosen AI provider (e.g., api.anthropic.com)
  4. AI provider processes the request and sends response
  5. Extension displays the response to you

‚ö†Ô∏è Critical: At no point in this flow does data pass through Apra Labs servers. The connection is direct: Your Browser ‚Üî AI Provider

5. Your Rights and Controls

You have complete control over your data:

‚úÖ What You Can Do

  • View Your Data - Check Chrome DevTools ‚Üí Application ‚Üí Storage ‚Üí Extensions
  • Export Settings - Use "Export Configuration" in Advanced tab

üîß How to Delete Your Data

Option 1: Clear Settings (keeps extension installed)

  1. Click AI Code Buddy icon in Chrome toolbar
  2. Go to "Settings" tab
  3. Click "Clear All Settings"
  4. Confirm deletion

Option 2: Uninstall Extension (complete removal)

  1. Go to chrome://extensions/
  2. Find "AI Code Buddy"
  3. Click "Remove"
  4. All data is permanently deleted

6. GDPR & CCPA Compliance

üá™üá∫ GDPR (European Users)

Under GDPR, you have the right to:

  • Access - View what data is stored (check Chrome storage)
  • Rectification - Edit your settings anytime
  • Erasure - Delete all data (clear settings or uninstall)
  • Portability - Export your configuration
  • Object - Stop using the extension anytime

Legal Basis for Processing: Consent - By installing and using AI Code Buddy, you consent to local storage of your settings.

üá∫üá∏ CCPA (California Users)

Under CCPA, you have the right to:

  • Know - We've disclosed all data collection above
  • Delete - Clear settings or uninstall the extension
  • Opt-Out - Don't install or use the extension

Note: Since we collect no personal information and have no servers, most GDPR/CCPA requirements don't apply to Apra Labs. However, they may apply to the AI provider you choose. Please review their privacy policies.
