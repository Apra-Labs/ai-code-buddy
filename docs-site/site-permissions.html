Site Permissions

Control where AI Code Buddy runs. Simple, powerful, and secure.

Allow Specific Sites Only

Perfect for work-only usage. Run AI Code Buddy only on your approved sites.

  1. Go to chrome://extensions/
  2. Find AI Code Buddy ‚Üí Click Details
  3. Under "Site access" ‚Üí Select On specific sites
  4. Click Add a new page
  5. Enter your site URLs (e.g., https://rport.company.com)
üí° Pro Tip: Use wildcards like https://*.company.com/* for all subdomains

Block Specific Sites

Run everywhere except sites you choose to block (social media, email, etc.).

  1. Visit the site you want to block
  2. Click the AI Code Buddy icon
  3. Right-click the icon
  4. Select: This can't read or change site data
  5. Refresh the page

Temporarily Disable

Need a break? Disable the extension completely without uninstalling.

  1. Go to chrome://extensions/
  2. Find AI Code Buddy
  3. Toggle the switch to OFF
  4. Toggle back to ON when needed
