Detailed Setup Guides

Google Gemini Recommended for Beginners

FREE Tier No Credit Card
▼

Get Your API Key

  1. Go to Google AI Studio
  2. Sign in with your Google account
  3. Click "Create API Key"
  4. Select or create a Google Cloud project
  5. Copy the key (starts with AIza)

Pricing

FREE 60 requests/minute
$0.50 Per million tokens (paid)

Why Gemini? Perfect for testing and learning. No credit card required for the free tier. Good quality responses with generous rate limits.

Cost Example

100 script improvements: FREE (typical usage stays within free tier)

Ollama (Local AI) 100% Private

100% FREE No API Key Offline
▼

Setup (Windows)

  1. Download from ollama.ai/download
  2. Run the installer
  3. Open PowerShell or Command Prompt
  4. Pull a model: ollama pull codellama
  5. Verify: ollama list

Setup (Mac/Linux)

  1. Install: curl -fsSL https://ollama.ai/install.sh | sh
  2. Pull a model: ollama pull codellama
  3. Verify: ollama list

Extension Configuration

  • Endpoint: http://localhost:11434
  • Model: Choose from your pulled models (e.g., codellama)
  • API Key: Not needed!

Why Ollama? Your data never leaves your machine. Completely free and private. Perfect for sensitive code and environments without internet access.

Recommended Models

CodeLlama Best for code (~4GB)
Llama 2 General purpose (~4GB)
Mistral Fast & efficient (~4GB)

System Requirements: 8GB RAM minimum (16GB recommended), 10GB+ free disk space per model

Claude (Anthropic) Best Quality

Best for Code Pay-as-you-go

Get Your API Key

  1. Go to console.anthropic.com
  2. Sign up or log in
  3. Click "API Keys" in the sidebar
  4. Click "Create Key"
  5. Name your key (e.g., "AI Code Buddy")
  6. Copy the key (starts with sk-ant-)

Prices per million tokens

OpenAI (GPT-4 / GPT-3.5)

Most Popular $5 Trial Credit
▼

Get Your API Key

  1. Go to platform.openai.com
  2. Sign up or log in
  3. Click your profile → "View API keys"
  4. Click "Create new secret key"
  5. Name your key
  6. Copy the key (starts with sk-)

Organization ID (Optional)

  1. Go to Organization Settings
  2. Copy your Organization ID (starts with org-)
  3. Paste in extension settings if needed

Pricing

$0.50 GPT-3.5 Turbo
$10 GPT-4 Turbo
$30 GPT-4

Prices per million tokens

Cost Example

100 script improvements with GPT-3.5: ~$0.05

100 script improvements with GPT-4: ~$1.50

New Users: Get $5 in free trial credits to test the service!

Azure OpenAI

Enterprise Requires Approval
▼

Prerequisites

  • Azure account with active subscription
  • Access to Azure OpenAI Service (requires approval)

Get Access

  1. Request access at aka.ms/oai/access
  2. Fill out the form with your use case
  3. Wait for approval (can take several days)

Setup After Approval

  1. Go to Azure Portal
  2. Create "Azure OpenAI" resource
  3. Note your resource name
  4. Deploy a model (e.g., gpt-35-turbo)
  5. Go to "Keys and Endpoint"
  6. Copy KEY 1 and Endpoint URL

Extension Configuration

  • API Key: Your 32-character Azure key
  • Endpoint: https://YOUR-RESOURCE.openai.azure.com
  • Deployment Name: Your model deployment name
  • API Version: 2023-05-15

Why Azure? Enterprise-grade security, compliance, and SLA. Best for corporate environments with strict data governance requirements.

Other Providers

Cohere HuggingFace Replicate
▼

Cohere

  1. Go to dashboard.cohere.ai
  2. Sign up or log in
  3. Click "API Keys" in sidebar
  4. Copy your trial key or create a production key

Pricing: Free trial, then $1-2 per million tokens

HuggingFace

  1. Go to huggingface.co
  2. Sign up or log in
  3. Go to Settings → Access Tokens
  4. Create new token with "Read" permission
  5. Copy token (starts with hf_)

Pricing: Free tier available, rate limits apply

Replicate

  1. Go to replicate.com
  2. Sign up or log in
  3. Go to Account → API Tokens
  4. Copy your token (starts with r8_)

Pricing: Pay per second of compute (~$0.0005/second)

Security Best Practices

DO:

  • Keep API keys private
  • Rotate keys regularly
  • Use environment-specific keys
  • Monitor usage in provider dashboards
  • Set spending limits

DON'T:

  • Share keys in code or screenshots
  • Commit keys to Git repositories
  • Use the same key everywhere
  • Share keys with others
  • Post keys in public forums
